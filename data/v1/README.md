BLSC — Version 1 (v1) Dataset Structure

This directory defines the official Version 1 (v1) dataset structure of the
Bulgarian Language & Script Corpus (BLSC).

Folders

raw/            – Original source texts, unmodified.
normalized/     – Cleaned and normalized versions of raw texts.
metadata/       – JSON / CSV metadata files for each document.
samples/        – Small example files for demonstration and testing.

Version Status

v1.0 — Architectural skeleton (initial structure)

Content will be added gradually according to the Roadmap (2025–2026)
