This folder contains structured metadata for each document in the dataset.
All metadata files must follow the official BLSC Metadata Model (see `Linguistic_Metadata_Model.md`).

Metadata filenames must match the base name of the corresponding raw and normalized files.
Example:
- raw/article001.txt
- normalized/article001.txt
- metadata/article001.json
