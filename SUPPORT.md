# Support • Bulgarian Language & Script Corpus (BLSC)

Thank you for your interest in the Bulgarian Language & Script Corpus!

This project is community-driven and maintained entirely by volunteers.  
Support is provided publicly and openly through the following channels:

---

##  1. Issues (Bug Reports & Questions)
If you have found a bug, need clarification, or want to propose an improvement,  
please open an issue here:

 **https://github.com/blsc-org/blsc-core/issues**

Before opening a new issue:
- Check if a similar issue already exists
- Provide clear and reproducible details
- Include dataset/module paths when relevant

---

##  2. Contribution Guidelines
Before contributing, please read:

 **CONTRIBUTING.md**

It describes:
- how to submit pull requests  
- formatting and metadata guidelines  
- dataset structure rules  
- legal and ethical restrictions
